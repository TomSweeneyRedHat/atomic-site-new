Atomic Host

Atomic Host (AH) is a container-based operating system that allows for secure updates and rollbacks. It provides immutable infrastructure for deploying to hundreds or thousands of servers in your private or public cloud. Available in Fedora Atomic Host, CentOS Atomic Host, and Red Hat Atomic Host editions depending on your platform and support needs.

To balance the need between long-term stability and new features, we are providing different releases of Atomic Host for you to choose from.

Atomic Workstation

Fedora Atomic Workstation (FAW) is the desktop version of Atomic Host - if you need a container-based operating system (OS) that will give you atomic rollbacks, you're in the right place. Immutable infrastructure for your desktop experience - if you want to know, what all this means, head to the documentation. Fedora Atomic Workstation is still heavily in development with only few external users.

This project is actively maintained and is ready for use by sophisticated and interested users, but not ready for widespread promotion.
