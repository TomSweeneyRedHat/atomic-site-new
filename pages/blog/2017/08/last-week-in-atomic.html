Last Week In Atomic

by Josh Berkus – Monday 14 August 2017

Since Project Atomic has now spread out across several blogs and websites, this is the second of what will become regular updates on events and posts around the Atomic community. This is also a useful catch-up for anyone who doesn’t have time to backfill on all of the individual blog posts.

This week includes kpod, vagrant-buildah, Fedora Atomic Host, CentOS Atomic Host, Kubernetes containers, and more.

New Projects and Tools

As a sub-project of CRI-O, that team has now announced development of the kpod command. kpod is an administrative and troubleshooting tool for containers running on individual kubernetes nodes, especially if the docker daemon is not involved. Current commands include version, push, and pull.

Tyler Britten released some scripts for playing with Buildah on Windows or Mac, using Vagrant.

Releases

  • Fedora released their August 8th Update for Fedora 26 Atomic.
  • CentOS released version 7.1707 of CentOS Atomic Host.
  • The CentOS Virt SIG and Atomic SIG released new Kubernetes system containers based on version 1.7.

Blog Posts

  • Dusty Mabe explained how the Fedora OSTree repos get built.
  • Tom Sweeney blogged about using Buildah to create fit images.
  • Joe Brockmeier lays out what will be in RHEL Atomic 7.4, coming soon.

Events

The Fedora Atomic WG will be participating heavily in the upcoming Flock to Fedora conference. This includes sessions on multi-architecture containers, becoming a container package maintainer, system containers, an Atomic Host documentation writeathon, and many more sessions.

ContainerCamp UK will feature Michael Hausenblas of the OpenShift team talking about Kubernetes backup.

OSS Summit NA will have an Atomic booth presence in the Red Hat pavillion. We could use booth volunteers if you love Atomic. We will also have a BOF on Atomic Host.

Kubecon US still has their CfP open. Sumbit talks involving Atomic projects!
