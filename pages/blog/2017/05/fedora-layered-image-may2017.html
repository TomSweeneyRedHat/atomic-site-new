Fedora Layered Image Release May 2017

by Adam Miller – Wednesday 24 May 2017

On behalf of the Fedora Atomic WG and Fedora Release Engineering, I am pleased to announce the latest Fedora Layered Image Release. This release includes many new images, especially a complete set of images for running Kubernetes in containers.

At this time the following Container Images are available in the Fedora Registry:

Base Images

(Note that the latest tag currently points to 25 and the rawhide tag currently points to 27, if no tag is provided in your pull command then it will always default to latest)

  • registry.fedoraproject.org/fedora:latest
  • registry.fedoraproject.org/fedora:rawhide
  • registry.fedoraproject.org/fedora:27
  • registry.fedoraproject.org/fedora:26
  • registry.fedoraproject.org/fedora:25
